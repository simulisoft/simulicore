Detailed Description

template<typename T_Matrix>
class cla3p::PartialQR< T_Matrix >

The partial QR decomposition class.

Utility class for the rank reduction process. Uses a pivoted QR decomposition to process a chunk of the input matrix until the stopping criterion is reached. Can be resumed on request, or perform a single step. The input matrix is automatically transposed if needed, in order for the decomposition to be applied on tall matrices (num rows >= num columns).

◆ resume()

template<typename T_Matrix >
void cla3p::PartialQR< T_Matrix >::resume ( bool ignoreTol = false)

Resumes the partial qr decomposition.

In case decompose() returned earlier than expected, set a new tolerance value and call resume() that will run until the new tolerance criterion |Ri,i| > |R0,0| * cutTolerance() is reached.
If the ignoreTol option is true, the decomposition will proceed until every column is processed.

Parameters
[in]ignoreTolThe decomposition will ignore the tolerance criterion and proceed to full decomposition.
