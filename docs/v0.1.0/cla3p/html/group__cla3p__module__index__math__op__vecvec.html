dns::XxMatrix< T_Scalar > & A )

Update a matrix with a vector outer product.

Performs the operation A = A + alpha * X YT
A can be General/Symmetric/Hermitian.
If A is Symmetric/Hermitian, it is assumed that the outcome of the operation X YT is also Symmetric/Hermitian and only the corresponding part (upper/lower) will be calculated.

Parameters
[in]alphaThe update coefficient.
[in]XThe input vector.
[in]YThe input vector.
[in,out]AThe matrix to be updated.
