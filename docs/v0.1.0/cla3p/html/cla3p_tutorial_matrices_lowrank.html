Low-rank Matrices

CLA3P supports the following low-rank matrices:

// using templates:
cla3p::lra::LrMatrix<cla3p::dns::RdMatrix> Ad; // double precision real low-rank matrix
cla3p::lra::LrMatrix<cla3p::dns::RfMatrix> As; // single precision real low-rank matrix
cla3p::lra::LrMatrix<cla3p::dns::CdMatrix> Az; // double precision complex low-rank matrix
cla3p::lra::LrMatrix<cla3p::dns::CfMatrix> Ac; // single precision complex low-rank matrix
// or using standard typedefs:
cla3p::lra::RdMatrix Ad; // double precision real low-rank matrix
cla3p::lra::RfMatrix As; // single precision real low-rank matrix
cla3p::lra::CdMatrix Az; // double precision complex low-rank matrix
cla3p::lra::CfMatrix Ac; // single precision complex low-rank matrix

Creating matrices

You can declare/create low-rank matrices in the following ways:

Default creation

CodeOutput
#include <iostream>
#include "cla3p/dense.hpp"
#include "cla3p/lra.hpp"
int main()
{
/*
* Double precision real empty low-rank matrix
*/
cla3p::lra::RdMatrix A;
std::cout << A.info("A");
/*
* Rank 2 (3x4) single precision real low-rank matrix (uninitialized values)
*/
cla3p::lra::XxMatrix<cla3p::dns::RfMatrix> B(3,4,2);
std::cout << B.info("B");
/*
* Allocate space for A (rank 2, 5x3, uninitialized values)
*/
A = cla3p::lra::RdMatrix(5,3,2);
std::cout << A.info("A");
return 0;
}
