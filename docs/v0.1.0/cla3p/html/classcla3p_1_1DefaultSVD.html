The dimensional constructor.

Constructs a svd object with internal buffer pre-allocation.

Parameters
[in]mThe maximum number of rows the buffer can fit.
[in]nThe maximum number of columns the buffer can fit.
[in]leftPolicyIf true, a (m x m) buffer will be allocated for U, otherwise a (m x min(m,n)).
[in]rightPolicyIf true, a (n x n) buffer will be allocated for V, otherwise a (n x min(m,n)).
